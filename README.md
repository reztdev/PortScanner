# PortScanner

Simple Port Scanner with nmap tools.

## Install nmap
- Ubuntu & Debian
```sudo apt-get install nmap```
- Arch
```sudo pacman -S nmap```
- Budgie
```sudo eopkg install nmap```
- Termux
```pkg install nmap```
- Windows go to
[`Nmap`](https://nmap.org/download.html#windows) and install `setup.exe` (as run Administrator)

## Install module
```
pip install python-nmap
```
